身体系统指南

Systematic Body Guide

头前引 (Text Neck)

现代病

耳垂位置明显超前于肩峰，通常伴随颈椎曲度变直。

Daily Cause / 日常诱因 视线长期低于水平线（手机/笔记本电脑），枕头过高。
Correction / 修正建议 将显示器垫高至视线平齐；采用“下巴回收”动作（Double Chin）进行深层颈屈肌训练。

视觉性偏转

习惯性

由于长期单侧用眼或双屏办公导致的面部/颈部不对称。

Daily Cause / 日常诱因 工位布局不合理，长期单侧咀嚼，斜视屏幕。
Correction / 修正建议 调整主辅屏位置至正前方；定期进行眼球米字操；左右手交替使用鼠标/手机。

上交叉综合症 (圆肩)

高频问题

胸肌紧张导致肩部内扣，背部肌群被拉长无力。

Daily Cause / 日常诱因 长期伏案打字，双臂前伸，胸椎灵活性受限。
Advice / 建议 多做开门拉伸（胸小肌放松）；强化菱形肌（W字夹背训练）；使用弹力带进行面拉（Face Pull）。

功能性高低肩

结构性

非病理性的侧弯，常表现为一侧肩部耸起，衣服吊带易滑落。

Daily Cause / 日常诱因 长期单肩背包，习惯性单侧托腮，睡觉习惯侧卧一侧。
Advice / 建议 避免二郎腿；单肩包改为双肩包；加强核心肌群两侧的平衡性，进行侧支撑训练。

骨盆前倾

伪翘臀

腹部突出，腰椎压力过大，久站后腰痛明显，睡觉平躺腰部悬空。

Daily Cause / 日常诱因 久坐导致髂腰肌缩短紧张，腹肌无力。
Training / 训练方向 拉伸髂腰肌（弓步拉伸）；强化腹直肌（死虫式）与臀大肌（臀桥）；平时注意收紧核心。

足弓塌陷 (扁平足)

根基问题

足弓失去弹性，导致膝盖内扣（X型腿）及步态异常。

Daily / 日常注意 避免穿底过软的鞋子；过度肥胖会增加足底压力。
Correction / 修正建议 尝试赤足在沙地/草地行走；进行“抓毛巾”训练激活足底肌群；练习单腿站立平衡。
